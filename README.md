# Quest Academy  - Learn. Play. Level Up.

### 📖 Project Overview
A mobile application designed to make learning interactive and rewarding. The Users complete quests, earn experience points (XP), unlock achievements, and progress through different levels, learning different CS categories. The application includes modern mobile UI/UX, gamification mechanics, and audio based learning experiences with reward systems.

## ✨ Features
1. 🏠 Home Screen
- Personalized greeting section
- User profile and level progress display
- XP progress tracking
- Carousel slider for top learning categories
- Achievement showcase section
- Grid based quest explorer
  
2. 👤 Profile Screen
- Full screen user profile interface
- Level and XP statistics
- Progress indicator
- Achievement tracking

3. 🎯 Quest System
- Detailed quest information page
- Quest difficulty indicators
- XP reward information
- 🎵 Audio Learning Experience
- Play/Pause functionality
- Automatic quest completion detection

4. 🏆 Reward System
- Animated congratulation screen
- XP reward display
- Badge unlocking system

## 📦 Package Usage
 1. carousel_slider: Display interactive category carousel
 2. just_audio: Audio playback and controls
 3. lottie: Reward and celebration animations

## 📱 Getting Started
1. Locate the APK at:
 - build/app/outputs/flutter-apk/app-release.apk
2. Transfer the APK file to your Android device.
3. Open the APK file.
4. Enable "Install from Unknown Sources" if prompted.
5. Tap Install.
6. Launch Quest Academy and Enjoy!
